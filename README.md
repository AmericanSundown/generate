# generate [![NPM version](https://badge.fury.io/js/generate.svg)](http://badge.fury.io/js/generate)

> Project generator, for node.js.

## Install globally with [npm](npmjs.org):

```bash
npm i -g generate
```

## CLI

```js

```

## Running tests

```bash
npm i -d && npm test
```
